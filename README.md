# El Nudo Gordiano: Análisis del Derroche Fiscal en Chile

Este repositorio contiene un motor de simulación estocástica para modelar cómo la ineficiencia estatal, el sobreprecio en obras públicas y la carga impositiva específica afectan el ingreso real del ciudadano chileno.

## Los 4 Pilares del Modelo
1. **El Fraude del Hormigón**: Sobreprecios en licitaciones municipales.
2. **Exacción Energética**: El impacto del impuesto específico y volatilidad del Brent.
3. **Hipertrofia Burocrática**: La pérdida de eficiencia por capas administrativas.
4. **Commoditización**: Dependencia de variables externas.

## Estructura
- `src/`: Código fuente del motor de simulación.
- `notebooks/`: Análisis visual y forense.
- `data/`: Resultados de las simulaciones en CSV.
